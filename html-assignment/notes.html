  • h1/h2/h3/h4/h5/h6: These tags are used to define the headings within an HTML file.
  • abbr: This tag is used to show an abbreviation or an acronym.
  • !DOCTYPE: This is a mantatory tag to use at the beginning of HTML files. It sends the information to the browser about what type of document to expect.
  • img: This tag is used to define an image within a file.
  • li: This is a tag that is used to define a list. Within the li tag is when you would use ol or ul, depending on the type of list that is being made.
